Maison Glow

A deployed storefront experience for clothes, skincare, and lifestyle products with clean browsing, category discovery, and a polished shopping interface.

Overview

Maison Glow presents a modern e-commerce interface focused on product browsing, category filtering, and a smooth user experience. The project shows my ability to build product-oriented interfaces and connect frontend flows with deployable web architecture.

Key Features

  • Product listing with category navigation
  • Filtering and sorting experience
  • Wishlist-style product actions
  • Responsive storefront layout
  • Clean product discovery interface
