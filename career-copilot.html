Career Copilot

An AI-powered assistant for job preparation, helping candidates work on letters, interviews, matching, reformulation, and activity tracking.

Overview

Career Copilot combines web application workflows with AI-assisted features for candidates. It organizes job preparation tasks into a practical portal for writing, improving, tracking, and preparing career materials.

Key Features

  • AI-assisted letter and CV reformulation
  • Interview preparation workflows
  • Candidate matching support
  • History and action tracking dashboard
  • Structured interface for career preparation

Technologies Used

React Django Hugging Face Vercel AI NLP API REST APIs
