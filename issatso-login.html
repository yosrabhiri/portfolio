Internships Management

A complete academic platform for managing internships, supervision, reports, offers, dashboards, notifications, and constraint-based defense timetable generation.

Overview

Internships Management is a full-stack internship and defense management system built with an Angular frontend and a Laravel REST API. It supports multiple academic roles, including students, teachers, internship supervisors, and administration workflows.

Key Features

  • Student and teacher registration with JWT authentication
  • Internship file creation, updates, validation, and history
  • Supervision requests with teacher responses and notifications
  • Report deposit, keyword search, and submission period control
  • Internship offers, assignment workflows, and dashboard analytics
  • Automatic defense timetable generation while taking multiple constraints into account

Technologies Used

Angular Laravel MySQL JWT Auth API REST Netlify
